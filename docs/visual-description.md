# JARVIS 全息核心视觉描述

## 整体概念

一个悬浮在空中的 AI 全息核心，整体并不是实体球体，而是由大量金橙色发光线条、粒子轨迹、断续电弧、扫描环和数据流组成的动态球形结构。

## 核心层

- 球体中心存在一个高亮核心，像微型恒星，发出极强的白金色光
- 中心最亮，外围逐渐透明
- 存在呼吸脉动动画

## 外围结构

- 外围并非完整外壳，而是无数高速运动的粒子轨迹围绕核心形成"半透明球状轮廓"
- 轨迹类型：
  - 断裂的弧线
  - 不完整的扫描轨道
  - 漂浮的数据流
  - 闪烁的电路路径
  - 高频运动的粒子残影

## 球体表面特征

- 全息投影感
- HUD科技感
- 数字化扫描结构
- 非实体能量态
- 电磁场环绕效果

## 轨道系统

- 外围存在多层不同倾角的旋转轨道，像行星环，但由发光粒子组成
- 部分区域粒子密集形成亮带，部分区域稀疏形成空洞
- 整个球体看起来并不对称，像"正在实时计算中的AI"

## 球体内部

- 发光电路线
- 数据节点
- 能量脉冲
- 短暂闪烁的光点

## 光效特点

- 中心最亮
- 外围逐渐透明
- 粒子存在运动拖尾
- 亮度不均匀
- 有局部过曝的能量爆点

## 整体风格

- 钢铁侠贾维斯
- 全息AI核心
- 未来操作系统
- 能量数据球
- 高级人工智能中枢

## 配色方案

- 主色调：金橙色 (#FFB300, #FF8F00, #FF6F00)
- 核心色：白金色 (#FFFFFF, #FFF8E1, #FFD54F)
- 辅助色：蓝青色（用于对比，如底部冷光）

## 状态响应

| 状态 | 视觉表现 |
|------|----------|
| idle | 缓慢漂浮，低亮度，低频脉冲 |
| waiting | 轻微脉冲，中等亮度 |
| thinking | 粒子聚拢、旋转加速，高亮度 |
| executing | 分层轨道模式，电路线亮起 |
| responding | 粒子向外扩散，中等亮度 |
| error | 粒子闪烁、变色（偏红），剧烈抖动 |
